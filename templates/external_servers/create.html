Инструкция по настройке Glances

Для мониторинга внешнего сервера через Glances:

  1. Установите Glances на сервер:
    2. pip install glances
  2. Запустите Glances в веб-режиме:
    3. glances -w
  3. Чтобы Glances запускался автоматически, настройте системный сервис.
  4. По умолчанию Glances использует порт 61208. Убедитесь, что этот порт открыт в брандмауэре.
